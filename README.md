## ✨ Overview

This is a **full-stack Food Delivery Web Application** that allows users to explore food items, add them to cart, and place orders seamlessly.

It includes:

* 👨‍💻 **User Interface (Frontend)**
* ⚙️ **Backend APIs**
* 🔐 **Admin Dashboard**

Designed with scalability and real-world use cases in mind.

---

## 🎯 Key Features

### 👤 User Side

* 🍕 Browse food items
* 🛒 Add to cart
* 📦 Place orders
* 📱 Fully responsive design

---

### 🔐 Admin Panel

* ➕ Add new food items
* ✏️ Update existing items
* ❌ Delete items
* 📊 Manage orders

### ⚙️ Backend

* RESTful APIs

## ⚙️ Installation & Setup

### 🔹 Clone Repository

```bash
git clone https://github.com/deepak62017/Food-Delivery-website.git
cd Food-Delivery-website
```

---

### 🔹 Install Dependencies

#### 📦 Backend

```bash
cd backend
npm install
```

#### 🎨 Frontend

```bash
cd frontend
npm install
```

---

### 🔹 Run the Application

#### 🚀 Start Backend

```bash
npm start
```

#### 🚀 Start Frontend

```bash
npm start
```
